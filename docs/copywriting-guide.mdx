### 什么时候需要空格？

#### 中文与英文之间

<p style={{ margin: 0 }}>✅ 正确：</p>

> 在 **LeanCloud** 上，数据存储是围绕 **AVObject** 进行的。

<p style={{ margin: 0 }}>🔴 错误：</p>

> 在**LeanCloud**上，数据存储是围绕**AVObject**进行的。

> 在 **LeanCloud**上，数据存储是围绕**AVObject** 进行的。

<p style={{ margin: 0 }}>✅ 完整的正确用法：</p>

> 在 **LeanCloud** 上，数据存储是围绕 **AVObject** 进行的。每个 **AVObject** 都包含了与 **JSON** 兼容的 **key-value** 对应的数据。数据是 **schema-free** 的，你不需要在每个 **AVObject** 上提前指定存在哪些键，只要直接设定对应的 **key-value** 即可。

:::caution 注意
英文单词前后均要留出空格。
:::

:::info 提示
**「豆瓣 FM」**等产品名词，需要按照官方所定义的格式书写。
:::

#### 中文与数字之间

<p style={{ margin: 0 }}>✅ 正确：</p>

> 今天出去买菜花了 **5000** 元。

<p style={{ margin: 0 }}>🔴 错误：</p>

> 今天出去买菜花了 **5000**元。

> 今天出去买菜花了**5000**元。

:::caution 注意
数字前后均要留出空格。
:::

#### 链接与正文之间

:::caution 注意
链接前后均要留出空格。
:::

### 什么时候不需要空格？

#### 数字与单位之间

<p style={{ margin: 0 }}>✅ 正确：</p>

> 我家的光纤入户宽带有 **10Gbps**，**SSD** 一共有 **10TB**。

<p style={{ margin: 0 }}>🔴 错误：</p>

> 我家的光纤入户宽带有 **10** **Gbps**，**SSD** 一共有 **20** **TB**。

:::caution 注意
数字与单位之间不需要空格。
:::

<p style={{ margin: 0 }}>✅ 正确：</p>

> 今天是 **233°** 的高温。

> 新 **MacBook** **Pro** 有 **15%** 的 **CPU** 性能提升。

<p style={{ margin: 0 }}>🔴 错误：</p>

> 今天是 **233** **°** 的高温。

> 新 **MacBook** **Pro** 有 **15 %** 的 **CPU** 性能提升。

:::caution 注意
度「°」／百分比「%」与数字之间不需要空格。
:::

#### 全角标点与其他字符之间

<p style={{ margin: 0 }}>✅ 正确：</p>

> 刚刚买了一部 **iPhone**，好开心！

<p style={{ margin: 0 }}>🔴 错误：</p>

> 刚刚买了一部 **iPhone** ，好开心！

### 如何解决空格问题？

**Microsoft** 有个 **`-ms-text-autospace`** 的 **CSS** 属性可以实现自动为中英文之间增加空白。不过目前并未普及，另外在其他应用场景，例如 **OS X**、**iOS** 的用户界面目前并不存在这个特性，所以请继续保持随手加空格的习惯。

## 标点符号

### 不重复使用标点符号

<p style={{ margin: 0 }}>✅ 正确：</p>

> 德国队竟然战胜了巴西队！

> 她竟然对你说「喵」？！

<p style={{ margin: 0 }}>🔴 错误：</p>

> 德国队竟然战胜了巴西队！！

> 德国队竟然战胜了巴西队！！！！！！！！

> 她竟然对你说「喵」？？！！

> 她竟然对你说「喵」？！？！？？！！

## 全角与半角

### 使用全角中文标点

<p style={{ margin: 0 }}>✅ 正确：</p>

> 嗨！你知道嘛？今天前台的小妹跟我说「喵」了哎！

> 核磁共振成像（**NMRI**）是什么原理都不知道？**JFGI**！

<p style={{ margin: 0 }}>🔴 错误：</p>

> 嗨! 你知道嘛? 今天前台的小妹跟我说 "喵" 了哎!

> 嗨!你知道嘛?今天前台的小妹跟我说"喵"了哎!

> 核磁共振成像 (**NMRI**) 是什么原理都不知道? **JFGI**!

> 核磁共振成像(**NMRI**)是什么原理都不知道?**JFGI**!

### 使用半角数字字符

<p style={{ margin: 0 }}>✅ 正确：</p>

> 这件蛋糕只卖 **1000** 元。

<p style={{ margin: 0 }}>🔴 错误：</p>

> 这件蛋糕只卖 **１０００** 元。

:::caution 注意
在设计稿、宣传海报中如出现极少量数字的情形时，为方便文字对齐，是可以使用全角数字的。
:::

### 英文中使用半角标点

<p style={{ margin: 0 }}>✅ 正确：</p>

> 乔布斯那句话是怎么说的？「**Stay hungry, stay foolish.**」

> 推荐你阅读《**Hackers & Painters: Big Ideas from the Computer Age**》，非常的有趣。

<p style={{ margin: 0 }}>🔴 错误：</p>

> 乔布斯那句话是怎么说的？「**Stay hungry，stay foolish。**」

> 推荐你阅读《**Hackers＆Painters：Big Ideas from the Computer Age**》，非常的有趣。
