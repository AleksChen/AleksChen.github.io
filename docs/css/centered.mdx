## 居中元素尺寸固定

### absoulte + 负 margin

```Css

.father {
    position: relative;
}
.son {
    position: absolute;;
    top: 50%;
    left: 50%;
    margin-left: -50px;
    margin-top: -50px;
}
```

### absoulte + calc

```Css

.father {
    position: relative;
}
.son {
    position: absolute;;
    top: calc(50% - 50px);
    left: calc(50% - 50px);
}
```

### table-cell

这种方式需要父元素的宽高都是确定的，才能保证子元素在父元素中垂直水平都居中。

```Css

.father {
    height: 500px;
    width: 500px;
    display: table-cell;
    vertical-align: middle;
text-align: center;
}
.son {
    display: inline-block;

}
```

## 居中元素尺寸不固定

### absoulte + margin auto

```Css

.father {
    position: relative;
}
.son {
    position: absolute;;
    top: 0;
    right: 0;
    bottom: 0;
    left: 0;
    margin: auto;
}
```

### absolute + transform

```css
.father {
  position: relative;
}
.son {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}
```

### flex

```html
<div class="father">
  <div class="son">123123</div>
</div>
```

```css
.father {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

### grid（兼容性存在问题）

```html
<div class="father">
  <div class="son">123123</div>
</div>
```

```css
.father {
  display: grid;
}
.son {
  align-self: center; /*设置单元格内容的垂直位置*/
  justify-self: center; /*设置单元格内容的水平位置*/
}
```

## 行内元素垂直水平居中

### line-height

```html
<div class="father">
  <div class="son">123123</div>
</div>
```

```css
.father {
  width: 500px;
  height: 500px;
  line-height: 500px;
  text-align: center;
}
.son {
  display: inline-block;
  vertical-align: middle;
  line-height: inherit;
}
```
